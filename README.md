# AETHER NODE SYSTEM

AETHER NODE SYSTEM is a terminal-first distributed systems and automation engineering laboratory designed to explore networking, orchestration, observability, and distributed execution using constrained heterogeneous hardware.

The laboratory is built around four devices:

- Laptop (Control Hub)
- DIR-819 Router (Network Backbone)
- Redmi 3S (Edge Node)
- H96 Max (Compute Worker Node)

AETHER focuses on learning systems engineering through practical experimentation in Linux, networking, automation, monitoring, and distributed computing.

The goal is not computational power.

The goal is systems mastery.
